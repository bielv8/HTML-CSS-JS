EVITANDO SER HACKEADO POR "07) Tornando os Iframes Seguros.html"

Para melhorar sua segurança, você pode usar:

  • SANDBOX="SANDBOX": Bloqueio total de transferência de dados para o outro site.
  • SANDBOX="NO-REFERRER": Bloqueia a referência de dados. Não permite a captura de dados (Modo anônimo).
  • SANDBOX="ALLOW-SAME-ORIGIN": Permite a transferência de dados somente se o site do IFRAME estiver no mesmo servidor que o seu.
  • SANDBOX="ALLOW-SAME-ORIGIN ALLOW-FORM": Permite apenas formulários (No mesmo servidor).
  • SANDBOX="ALLOW-SCRIPTS": Permite a execução apenas de Scripts (JavaScript, PHP...).
